# Desafio para processo seletivo Verotecnologia 2021/22

## Dependências :
  - sass
  - sweetalert
  - react-icons
  - pdfmake
## Para a instalação:
  no terminal: 
  - Descer para a página desafio com "cd verotecnologia"
  - Instalar todas as dependências com "npm install"
  - E rodar o script do react com "npm start"
